🍔 Nutrients in Menu Items Dashboard
📌 Project Overview
This project analyzes the nutritional composition of restaurant menu items using an interactive dashboard built with Microsoft Power BI.
The dashboard provides insights into how different menu categories contribute to calorie intake and daily nutritional values. It helps identify which food categories contain higher calories, fats, vitamins, and minerals.
The analysis covers 260 menu items categorized into different food groups such as breakfast, beverages, desserts, and main meals.
🎯 Project Objectives
The main objectives of this analysis are:
Analyze calorie distribution across menu categories
Identify categories with the highest nutritional values
Compare vitamins and minerals contribution by food type
Understand the distribution of menu items across categories
Provide insights that support healthier menu decisions

| Vitamin C (%DV)     | Vitamin C daily value

📊 Key Business Insights
From the analysis we can conclude:
1️⃣ The menu is heavily dominated by beverage-related items.
2️⃣ Main meal categories such as Chicken & Fish have the highest calorie values.
3️⃣ Beef & Pork items contribute the highest iron levels, which may benefit customers needing iron-rich meals.
4️⃣ Salads provide higher fiber and Vitamin A, making them healthier options.
5️⃣ Many categories contain high saturated fat levels, indicating potential opportunities for healthier menu options.
